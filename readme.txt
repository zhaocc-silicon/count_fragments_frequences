Before running, you should pay an attention to:
a:You need modules pymysql
b:You should change the information of "get_mole_num_baba()" function in "Read_DB.py" according yourself database.

You can do these things:
1: run> python count_ele_sec_freq.py freq ele  # get the frequencies of elementary fragments
2: run> python count_ele_sec_freq.py freq sec # get the frequencies of secondary fragments
3: run> python count_ele_sec_freq.py atoms_num ele #get the heavy atomic number of all elementary fragments
4: run> python count_ele_sec_freq.py atoms_num sec # #get the heavy atomic number of all secondary fragments
